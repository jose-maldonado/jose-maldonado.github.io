Nick Doiron

Title: The Encrypted API: securing data with HTTPS and Paillier cryptosystem

Audience: Beginner

Description

This talk covers two tools for securing your users' data while developing a public API. (1) getting a free HTTPS cert with LetsEncrypt's Python CLI (2) the Paillier cryptosystem - develop APIs which can work on users' encrypted data, without accessing the unencrypted numbers. As an example, you could develop an exchange-rate API without sharing transaction details.

Who is Nick Doiron?

Nick is a traveling web developer and mapmaker. In the past he has worked with Code for America, One Laptop per Child, the City of Ulaanbaatar, and the Museum of Modern Art (NYC). His next project is post-quantum encryption for your e-mail.
