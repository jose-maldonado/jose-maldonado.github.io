Sam Clarke

Title: Using Celery in Production

Audience: All

Description

Celery is an asynchronous task queue for Python. This talk will start with a brief introduction to Celery, how and when to use it in your project and a discussion of common scenarios and pitfalls of using celery in Production. We will explore typical configurations of Celery in a production environment, how to keep Celery running happily and how to quickly diagnose failures. The talk will consist of: * Quick introduction to distributed task queues * Configuring Celery for Production - general principals * Adding asynchronous tasks to your code * Storing and Retrieving Task results * Monitoring & Testing * Scaling Celery
